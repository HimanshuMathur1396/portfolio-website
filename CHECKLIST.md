## üìù Customization Steps

### 1. Personal Information
```html
<!-- In index.html, update: -->
- Replace "Your Name" with your actual name
- Update email: your.email@example.com
- Update phone: +1 (555) 123-4567
- Update location: Your City, Country
- Update social media links
```

### 2. Content Updates
```javascript
// In script.js, update typing animation:
const texts = [
    'Your Title 1',
    'Your Title 2',
    'Your Title 3',
    // Add your roles
];
```

### 3. Skills & Projects
- Update skill percentages in HTML
- Replace project information
- Update technology tags
- Add your project links

### 4. Images Required
Create and add these files to the `assets` folder:
- `favicon.ico` (16x16 or 32x32 px)
- `profile.jpg` (300x300 px, square)
- `about-image.jpg` (400x300 px)
- `project1.jpg` (350x200 px)
- `project2.jpg` (350x200 px)
- `project3.jpg` (350x200 px)
- `project4.jpg` (350x200 px)

### 5. Color Customization
```css
/* In styles.css, update these variables: */
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
    --accent-color: #your-color;
}
```

## üåê Deployment Options

### GitHub Pages (Free)
1. Create GitHub repository
2. Upload all files
3. Go to Settings > Pages
4. Select source branch
5. Your site will be at: `username.github.io/repository-name`

### Netlify (Free)
1. Sign up at netlify.com
2. Drag and drop your folder
3. Get instant deployment
4. Custom domain available

### Vercel (Free)
1. Sign up at vercel.com
2. Connect GitHub repository
3. Auto-deployment on changes
4. Custom domain available

## üìä Performance Tips

### Image Optimization
- Compress images (use TinyPNG or similar)
- Keep file sizes under 500KB
- Use modern formats (WebP when possible)

### Loading Speed
- Minify CSS and JavaScript for production
- Use CDN for external resources
- Enable browser caching
