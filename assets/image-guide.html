How to Add Your Images

  1. Prepare your images in the recommended sizes
  2. Name them exactly as shown above
  3. Save them in the assets folder
  4. Use JPG format for photos, PNG for graphics with transparency

Image Optimization Tips

  • File Size: Keep images under 500KB each
  • Quality: Use 80-90% JPEG quality
  • Format: JPG for photos, PNG for graphics
  • Alt Text: Already included in the HTML
